This is a library management system.

## Tech Stack

- [Next.js 16](https://nextjs.org)
- React JS 19
- Tailwind CSS v4
- Better Auth
- Prisma
- PostgreSQL (Neon)

## Current Core Features
- Dashboard (View statistics and details)
- View, manage, and Search books and members
- Transactions (Lending and Returns)
